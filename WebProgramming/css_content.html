css 삽입법

  • 태그 내부 삽입형

    • <태그 style="속성:값; 속성:값;">

    • 태그의 스타일 속성으로 해당 태그에만 스타일을 적용하는 방식
    • 속성과 값은 콜론(:)으로 구분
    • 속성과 속성은 세미콜론(;)으로 구분

  • 문서 내 지정형

    • <style type="text/css">

        스타일 내용

    </style>

    • <style> 태그로 지정한 스타일이 현재 작업 문서 전체에 영향을 준다.

  • 외부 문서 삽입형

    • <link rel=stylesheet href="외부 css파일 경로" type="text/css">

    • <link> 태그로 외부에 있는 css 파일을 연결하여 여러 문서에 하나의 서식을 적용한다.
    • <link> 태그는 <head> 태그 사이에 입력한다.

선택자

스타일을 적용할 대상으로, { } 앞에 적어주는 것.

<style type="text/css">

    선택자{속성:값}

</style>
  • 태그 선택자

    • 태그{속성:값; 속성:값;}

  • 클래스 선택자

    • <style type="text/css">

        .클래스명{속성:값;}

    </style>

    <body>

        <태그 class=클래스명>

    </body>

    • 선택자를 사용할 때 클래스명 앞에 .을 붙인다.
    • 반복적으로 사용하는 스타일을 지정할 때 사용된다.
    • 어느 태그에나 사용할 수 있다.

  • 아이디 선택자

    • <style type="text/css">

        #아이디명{속성:값;}

    </style>

    <body>

        <태그 id=아이디명>

    </body>

    • 레이어를 동적으로 제어할 때 사용한다.
    • 주로 div 태그와 함께 사용된다.
    • 하나의 태그에만 적용된다.

  • 복수 선택자

    • 태그1, 태그2{속성:값; 속성:값;}

    • 여러 태그에 같은 스타일을 적용한다.

  • 자손 선택자

    • 태그1 태그2{속성:값;}

    • 지정된 태그 안에 포함된 태그에만 스타일을 적용한다.

  • 가상 선택자

    • a:지정자{속성:값;}

    • 실제로 현재 html문서상에 존재하지 않는 형식을 지정할 때 사용한다.
    • 주로 하이퍼링크 설정에 사용한다.
